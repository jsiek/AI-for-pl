## Syntax/Grammar

    Types         A, B, C  ::=  ι | X | α | A → B | ∀X.A | ★
    Ground Types  G, H     ::=  ι | α | ★→★
    Coercions     c, d     ::= id | G? | G! | X- | X+ | c → d | c ; d
                             | ∀X.c | 𝒢 | ℐ X.A

## Coercion Typing

    -----------------
    | Σ ⊢ c : A ⇒ B |
    -----------------

    Σ ⊢ A
    --------------
    Σ ⊢ id : A ⇒ A

    Σ ⊢ G
    ---------------
    Σ ⊢ G? : ★ ⇒ G

    Σ ⊢ G
    ---------------
    Σ ⊢ G! : G ⇒ ★
        
    --------------- (α=A ∈ Σ)
    Σ ⊢ α- : A ⇒ X

    --------------- (α=A ∈ Σ)
    Σ ⊢ α+ : X ⇒ A

    -----------------
    Σ ⊢ 𝒢 : A ⇒ ∀X.A

    ---------------------------
    Σ ⊢ ℐ X.A : ∀X.A ⇒ A[X:=★]

    Σ, X ⊢ c : A ⇒ B
    ------------------------
    Σ ⊢ ∀ X.c : ∀X.A ⇒ ∀X.B

    Σ ⊢ c : C ⇒ A   Σ ⊢ d : B ⇒ D
    ------------------------------
    Σ ⊢ c → d : A → B ⇒ C → D

    Σ ⊢ c : A ⇒ B   Σ ⊢ d : B ⇒ C
    ------------------------------
    Σ ⊢ c ; d : A ⇒ C
